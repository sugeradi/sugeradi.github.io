遇到如题描述的问题，可以尝试一下步骤：

1、更新conda到最新版本：
```shell
conda update -n base conda
```

![conda update -n base conda](https://www.pianshen.com/images/532/428adfa7e2d86dbd0e1399f30c3add94.png)

2、然后执行更新：
```shell
conda update --all
```
![](https://www.pianshen.com/images/80/e91935c3d9ceabadb6fd2015b904e8f8.png)

开始更新:
![](https://www.pianshen.com/images/461/094015f860ac7e854b91943cf1130bd5.png)

3、再次安装包
```shell
conda install -c conda-forge xxx
```
