Markdown支持常见的编程语言代码块高亮显示，但是一直以来不太清楚究竟支持哪些语言，如何进行标识？

以常用的python语言为例：

下面是一个批量重命名的代码
```txt
import os
path = 'D:\mydocuments\'
for file in os.listdir(path):
    if os.path.isfile(os.path.join(path,file))==True:
        if file.find('.')<0:
            newname=file+'rsfdjndk.jpg'
            os.rename(os.path.join(path,file),os.path.join(path,newname))
            print file,'ok'
#        print file.split('.')[-1]
```

在Markdown中显示为：

```python
# -*- coding: cp936 -*-
import os
path = 'D:\mydocuments\'
for file in os.listdir(path):
    if os.path.isfile(os.path.join(path,file))==True:
        if file.find('.')<0:
            newname=file+'rsfdjndk.jpg'
            os.rename(os.path.join(path,file),os.path.join(path,newname))
            print file,'ok'
#        print file.split('.')[-1]
```
